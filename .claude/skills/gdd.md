# /gdd — Generate a Game Design Document

**Usage**: `/gdd <title> [genre] [player_class] [level_count]`

**Description**: Generate a structured Game Design Document for a VoxelForge game.

## Template Output

When invoked, produce a GDD following this structure and save to `design/gdds/<title>.md`:

```markdown
# Game Design Document: <TITLE>

**Version**: 1.0
**Date**: <today>
**Genre**: <genre>
**Studio**: VoxelForge AI Game Studios

---

## 1. Executive Summary
One paragraph pitch: what the game IS, who it's FOR, and why it's FUN.

## 2. Target Audience
- **Primary Player Type**: [Achiever / Explorer / Socializer / Killer]
- **Age Range**: <range>
- **Experience Level**: <casual/mid-core/hardcore>
- **Play Session**: <5-15 min / 30-60 min / 2+ hours>

## 3. Core Loops

### 30-Second Loop (immediate)
> What the player does moment-to-moment

### 5-Minute Loop (session)
> A complete mini-challenge cycle

### 30-Minute Loop (progression)
> How the player advances toward a goal

## 4. MDA Framework

### Mechanics (rules)
- [ ] Movement: WASD + Space (jump)
- [ ] Interaction: E key (collect/open)
- [ ] Combat: proximity-based (enemy aggro)
- [ ] Objective: collect all chests

### Dynamics (emergent)
- Players path-find around enemies to reach chests
- Risk/reward: more enemies → more tension → more satisfaction on collection

### Aesthetics (experience)
- Primary: **Challenge** (overcoming enemies)
- Secondary: **Discovery** (finding hidden chests)
- Tertiary: **Sensation** (pixel-art isometric voxel style)

## 5. Level Design

| Level | Genre | Size | Enemies | Objective |
|-------|-------|------|---------|-----------|
| 1     | <genre> | 24×24 | 1 | Tutorial: find 1 chest |
| 2     | <genre> | 32×32 | 2 | Find 2 chests |
| 3     | <genre> | 48×48 | 4 | Boss area, 3 chests |

## 6. Character Design

### Player
- **Class**: <player_class>
- **Speed**: <10-14> units/s
- **Jump**: <7-9> units
- **Starting HP**: 100

### Enemies
- **Class**: <enemy_class>
- **Aggro Distance**: <10-15> units
- **Patrol Behaviour**: random direction change every 3s
- **HP**: 30

## 7. Asset List

Generated by VoxelForge:
- [ ] Level geometry (terrain/dungeon)
- [ ] Player character (.vox)
- [ ] Enemy characters (.vox × N)
- [ ] Props (.vox × M)
- [ ] Player Lua script
- [ ] Enemy AI scripts (×N)
- [ ] Objective script

## 8. Technical Scope

```python
from forge.generators.game import GameGenerator
gen = GameGenerator(Palette.natural(), seed=42)
manifest = gen.generate(
    title        = "<TITLE>",
    genre        = "<genre>",
    player_class = "<player_class>",
    enemies      = <N>,
    props        = <M>,
    level_size   = <size>,
)
```

## 9. Win / Loss Conditions

- **Win**: All chests collected
- **Loss**: HP drops to 0
- **Retry**: Restart from scene beginning

## 10. Milestones

| Sprint | Goal |
|--------|------|
| S1     | Level + player movement |
| S2     | Enemies + combat |
| S3     | Objectives + HUD |
| S4     | Polish + release |
```
